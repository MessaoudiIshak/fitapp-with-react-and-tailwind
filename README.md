This app helps you:

- Generate customized workouts based on your preferences
- Choose between different workout splits (Individual, Bro Split, Bodybuilder Split, Upper/Lower)

## How It Works

1. **Pick Your Poison**: Choose your preferred workout split
   - Individual (target specific muscles)
   - Bro Split (Push/Pull/Legs)
   - Bodybuilder Split (body part focused)
   - Upper/Lower Split

2. **Lock On Targets**: Select up to 3 muscle groups to focus on
   - Multiple muscle selection for individual workouts
   - Pre-defined muscle groups for other splits

3. **Set Your Goal**: Choose your training objective
   - Strength/Power
   - Growth/Hypertrophy
   - Cardiovascular Endurance

The app will generate a complete workout with:
- Exercise selection and order
- Rep ranges or durations
- Rest periods between sets
- Tempo guidelines for each movement
- Set tracking (0-5 sets per exercise)

## Run the App Locally

1. Clone the repository and navigate into the project directory
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser to the URL shown in the terminal (commonly http://localhost:5173)
